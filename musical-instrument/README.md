## Aurora Piano — интерактивный музыкальный инструмент (React + TypeScript + Vite)

Красивый, адаптивный и интерактивный синтезатор‑пианино в браузере с плавными анимациями, визуализатором звука и настраиваемыми параметрами: волна, ADSR, фильтр, реверб.

Почему JavaScript/React/WebAudio:
- Современный UI и анимации удобнее всего строить в вебе.
- WebAudio API обеспечивает низкую задержку, полифонию, фильтры и эффекты.
- Vite + React + TypeScript дает быстрый DX, типовую безопасность и модульную архитектуру.

### Возможности
- Полифонический синтезатор (sine/square/saw/triangle)
- ADSR огибающая на каждую ноту
- НЧ‑фильтр с настраиваемой частотой среза и Q
- Процедурный реверб с регулируемым уровнем
- Визуализатор: столбцы спектра и «сияние»
- Игра мышью, клавиатурой (QWERTY) и тачем
- Адаптивная верстка, градиенты и стеклянные панели

### Установка и запуск
```bash
cd /workspace/musical-instrument
npm install
npm run dev
# сборка
npm run build && npm run preview
```

Откройте адрес из терминала (обычно `http://localhost:5173`). Нажмите «Включить звук» для инициализации аудио (требуется действие пользователя).

### Структура кода
- `src/audio/AudioEngine.ts` — движок WebAudio (огибающая, фильтр, реверб, анализатор)
- `src/context/AudioEngineContext.tsx` — провайдер состояния и методы управления
- `src/components/Studio.tsx` — композиция интерфейса
- `src/components/Visualizer.tsx` — канвас‑визуализатор
- `src/components/ui/Controls.tsx` — регуляторы параметров
- `src/components/piano/Piano.tsx` — клавиши, тач/клики и клавиатура
- `src/hooks/usePianoInput.ts` — раскладка клавиатуры и активные ноты
- Стили: `studio.css`, `controls.css`, `piano.css`, глобальные — `index.css`

### Управление
- Кнопка «Включить звук» — запуск аудиоконтекста
- Селектор волны, ADSR слайдеры, фильтр (Cutoff/Q), реверб (Wet/Length/Decay)
- Октава: диапазон от −2 до +2
- Играйте мышью по клавишам или с клавиатуры (например, `Q W E R T Y U` и нижний ряд `Z X C ...`)

### Примечания
- Используйте наушники для лучшего опыта.
- На мобильных устройствах аудио может требовать жестов пользователя для старта.
